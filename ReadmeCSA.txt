# 📊 Customer Sales Analysis – Excel Assignment

## 🔍 Project Description
This project is a hands-on Excel assignment where I analyzed a customer sales dataset using built-in Excel **statistical functions** and **pivot tables**.

## ✅ Excel Functions Used:
- **SUMIF**, **SUMIFS**
- **COUNTIF**, **COUNTIFS**
- **AVERAGEIF**, **AVERAGEIFS**
- **MAXIFS**

## 📌 Insights Extracted:
- Total transactions by State and Product Category
- Total quantity sold by Region and Product Name
- Maximum sales by City
- Average sales by State and Product

## 🧠 Skills Demonstrated:
- Conditional aggregation using formulas
- Report structuring in Excel
- Conversion of Excel report to PDF
